✺ TECHNOLOGY PREPARATION GUIDE

Mastering the tools for the TMX Cybersecurity Architect role.

SOC MONITORING & SIEM

Fundamental across TMX's Information Security Office.

ELK Stack: ElasticSearch (Storage), Logstash (Parser), Kibana (Visualization). centralize all log telemetry.
Splunk: High-performance log parsing and searching. Used for massive datasets in financial sectors.

ENDPOINT PROTECTION (EDR)

Securing work-from-home and server endpoints.

EDR Tools: Standard across the board for real-time memory and process monitoring (e.g., CrowdStrike, SentinelOne).
Antivirus: Traditional signature-based detection, supplemented by EDR's behavioral analysis.

NETWORK DEFENSE (IDS/IPS & FIREWALLS)

Controlling the perimeter for trade security.

IDS/IPS: Monitoring and blocking malicious packet headers. Snort is a key open-source reference.
WAF: Web Application Firewall. Protects trade portals from SQL Injection and XSS at Layer 7.

CLOUD SECURITY (AWS/AZURE)

Monitoring TMX's modernized cloud footprint.

CloudTrail: The definitive audit log for AWS. Shows exactly who made every API call.
GuardDuty: Managed threat detection using machine-learning to detect anomalies like Bitcoin mining or unauthorized access.
