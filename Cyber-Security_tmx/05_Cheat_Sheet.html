Quick Reference for Essential SOC Concepts and Commands.

NIST IR LIFECYCLE

  • Preparation: Tools, Policies, Training.
  • Detection & Analysis: Monitoring, Triage.
  • Containment/Eradication: Stopping the threat.
  • Recovery: Restoring operations.
  • Post-Incident: Lessons Learned.

SIEM LOGIC (ELK / SPLUNK)

  • Grok/Parsing: Normalizing raw logs into structured data.
  • Correlation: Identifying patterns across different logs.
  • Thresholds: Defining what triggers an alert (e.g., >5 fails/min).

ESSENTIAL PORTS

  • FTP: 21
  • SSH: 22
  • Telnet: 23
  • DNS: 53 (UDP)
  • HTTP/HTTPS: 80/443
  • RDP: 3389

PYTHON ANALYZER SNIPPET

import re
log = "Failed password for root from 192.168.1.1 port 22"
match = re.search(r"Failed.*from ([\d\.]+)", log)
if match:
    print(f"Malicious IP detected: {match.group(1)}")
